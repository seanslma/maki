# vim

## 5.1 execute command
```
:!cmd ->execute cmd
:!ls  ->execute ls
```

## 6.5 set option
```
:set paste
:set nopaste
:set ic     ->ignore case
:set noic   ->disable ignoring case
:set hls is ->highlight search and incremental search
:nohlsearch ->remove highligting of searches
/ignore\c   ->ignore case only in current search
:set number ->line number
:set nonumber
```

## syntax highlight
put it in .vimrc
```
:sy enable
:syn enable
:syntax enable
```

## 7.1 online help
```
<F1> key
:help
CTRL-W CTRL-W   ->to jump from one window to another
:q              ->close help window
:help w
:help c_CTRL-D
:help insert-index
:help user-manual
```
