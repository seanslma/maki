# Shortcuts

## select current word
```
Ctrl + D
Ctrl + Shift + Arrow #select more
```

## select current line
```
Ctrl + L
```

## delete line
```
Ctrl  + Shift + K
```

## insert line
```
Ctrl + Enter          #insert below
Ctrl + Shift + Enter  #insert above
```

## comment/uncomment code
```
Ctrl + /
```

## column selection
```
Ctrl + Shift + Alt + Down/Up/left/Right/PageUp/PageDown
```

## go to line
```
Ctrl + G
```
