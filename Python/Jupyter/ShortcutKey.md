# Shortcut key
https://towardsdatascience.com/optimizing-jupyter-notebook-tips-tricks-and-nbextensions-26d75d502663

## cell to markdown
  Esc + m
  
## markdown to cell
  Esc + y

## insert cell above
  Esc + a

## insert cell bellow
  Esc + b

## split cell
  Ctrl + Shift + - 

## merge bellow
  Shift + m  

## comment/uncomment block
  Ctrl + /
